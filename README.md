# 🏎️ NEURO DRIVE — 보고, 실험하고, 대결하는 AI 리터러시 실험실

### 🎮 지금 플레이: **[sdg331.github.io/neuro-drive](https://sdg331.github.io/neuro-drive/)**

**Deep Q-Network(DQN)를 외부 라이브러리 없이 순수 JavaScript로 완전 구현한 강화학습 교육 게임.**

`index.html` 파일 하나로 동작합니다. 브라우저에서 열면 바로 실행됩니다 (인터넷·설치 불필요, 모바일 지원).
웹 서버에 올리면 **PWA**로 동작합니다 — 오프라인 캐시 + 홈 화면 설치.

## 왜 만들었나 — 문제 정의

**모두가 AI를 쓰는 시대에, AI가 "어떻게 배우는지" 본 사람은 거의 없습니다.**

1. **AI 리터러시의 공백** — 챗봇을 쓰는 것과 AI를 이해하는 것은 다릅니다. 교과서의 강화학습은 수식(벨만 방정식)뿐이고, 진짜 AI는 서버 뒤에 숨어 있습니다. 학습 과정을 직접 보고 만질 수 있는 도구가 없습니다.
2. **AI 안전 개념의 조기 교육 부재** — "AI에게 목표를 잘못 주면 어떤 일이 생기는가"(보상 해킹, 정렬 문제)는 AI 시대의 시민이 알아야 할 핵심 개념이지만, 이를 체험으로 가르치는 도구는 사실상 없습니다.
3. **진입 장벽** — 강화학습 실습에는 Python, GPU, 라이브러리 설치가 필요합니다. 교실에서는 불가능한 조건입니다.

**NEURO DRIVE의 답:** 설치 없는 HTML 파일 하나에, 진짜 DQN이 눈앞에서 학습하는 전 과정 + 현실 AI 문제를 재현하는 가이드 실험 + 결과를 증빙하는 자동 리포트를 담았습니다. 수식을 몰라도 10분이면 "AI가 배운다"는 것이 무엇인지 몸으로 알게 됩니다.

### 누구를 위한 것인가

| 대상 | 활용 |
|---|---|
| 중·고교 정보/과학 수업 | 2022 개정 교육과정 고교 「인공지능 기초」의 기계학습·강화학습 단원 실습 도구. 교사는 준비물 없이 링크 하나로 수업 |
| 대학 입문 강의 | DQN의 모든 구성요소(리플레이 버퍼, 타깃 네트워크, Double DQN)가 시각화된 살아있는 교보재 |
| 일반 학습자 | 게임 → 도전과제 → 실험 → 리포트로 이어지는 자기주도 학습 |
| 교실 AI 리그 | 각자 키운 모델을 파일로 제출 → ⚔️ AI vs AI 토너먼트. 훈련 전략(커리큘럼, 보상 설계)이 승부를 가름 |

## 🔬 실험실 — 현실의 AI 문제를 게임 안에서 재현

이 제품의 핵심. 각 실험은 **가설 → 실험 → 자동 측정 → 결론** 구조이며, 게임이 단계 달성을 자동 감지하고 실측 데이터로 결론을 작성합니다.

| 실험 | 재현하는 현실 문제 |
|---|---|
| 🧭 탐험 vs 활용 | 유튜브 추천의 필터버블, 신약 탐색 — 탐험-활용 딜레마. 그리디 평가로 정책의 진짜 실력을 측정하고, 결과가 가설과 다르면 밀집/희소 보상의 차이까지 가르침 |
| 🕳️ 보상 해킹 | SNS 체류시간 보상의 부작용, OpenAI CoastRunners 사례 — **AI 안전(정렬) 문제의 축소판** |
| 🪜 커리큘럼 러닝 | 쉬운 과제 → 어려운 과제 사다리 (OpenAI 로봇 손, 알파스타의 훈련법) |
| 🌍 일반화 | 자율주행이 모든 도로를 암기할 수 없는 이유 — 직접 그린 트랙에서 무학습 완주 시험 |
| ⚔️ 인간 vs 기계 | 기계의 일관성 vs 인간의 적응력 — 직접 대결로 체험 |

완료한 실험은 **📄 실험 리포트**(HTML, 인쇄/PDF 가능)로 내보내 수업 제출·대회 증빙에 쓸 수 있습니다. 리포트에는 학습 곡선·신경망 상태 이미지, 실측 결론, 도전과제·랩 기록이 자동 포함됩니다.

|---|---|
| 🧠 실시간 학습 관전 | 무작위로 벽에 박던 차가 1~2분(MAX 속도) 만에 완주하는 전 과정 |
| 🗺 **가치 지도 (V(s) 히트맵)** | AI가 트랙 각 지점에 매긴 가치를 색으로 오버레이 — "AI의 머릿속"을 직접 봄 |
| 🔬 신경망 내부 시각화 | 뉴런 활성화·가중치·선택된 행동을 매 프레임 렌더링 |
| 🎛 보상 설계 실험실 | 슬라이더로 보상 함수를 직접 설계 → 보상 해킹 유발 |
| 🗺 트랙 3종 + 에디터 | 오벌(초급)/클래식(중급)/서펜트(고급) + 손으로 그리는 커스텀 트랙, 공유 코드 |
| 🏁 AI와 대결 | 3바퀴 레이스, 충돌 스턴, 시뮬레이션 틱 기반 공정 랩타임, 트랙별 기록 |
| ⚔️ AI vs AI 관전 | 내 AI vs 내장 챔피언 or 친구의 모델 파일 |
| 🏆 **AI 리그** | 모델 파일 여러 개로 **라운드 로빈 토너먼트** — 순위표·승점, 리포트에 자동 포함. 교실 AI 리그가 제품 안에서 완결 |
| 👻 **고스트 랩** | 내 최고 1랩 주행이 유령 차로 재생 — 트랙별 자동 저장·갱신 |
| 📅 **오늘의 트랙** | 날짜 시드로 매일 새 서킷 자동 생성 (서버 없는 일일 콘텐츠). 휴리스틱 봇이 **완주 가능성을 자동 검증** 후 제공 |
| ⚗️ **알고리즘 벤치마크** | 같은 초기 가중치·트랙·예산으로 기본 DQN vs Double vs PER vs 풀옵션(+3-step)을 자동 경쟁 — 학습 곡선 실시간 오버레이 + 실측 결론 + 리포트 수록. 내 모델은 완전 분리 보호 |
| 🌵 희소 보상 모드 | 체크포인트에서만 보상 — 탐험 난제를 극한 체험 (n-step의 가치를 몸으로 확인) |
| 🎯 도전과제 9종 | 학습 개념이 곧 게임 목표 (벤치마크 완주 = ⚗️ 알고리즘 과학자) |
| 🚀 온보딩 투어 | 첫 실행 시 스포트라이트 가이드 5단계 (설정에서 다시 보기) |
| 🎮 **게임 필 (v4.0)** | 네온 서킷(글로우 벽·연석·스타필드, 오프스크린 프리렌더) · 스키드 마크 · 부스터 화염 · 충돌 화면 흔들림+히트스톱+붉은 비네트 · 속도계 게이지 · P1/P2+격차 · FINAL LAP/NEW RECORD 배너 · 승리 콘페티 세리머니 |
| 🕹 어트랙트 모드 | 타이틀 화면 뒤에서 사전 학습 AI가 실제 주행 (오락실 감성) + 네온 로고 |
| 🎵 BGM | Web Audio 신스 시퀀서로 실시간 합성하는 칩튠 루프 (에셋 0, 설정 토글) |
| 🎓 사전 학습 모델 내장 | 70,000스텝 학습된 가중치 포함 — 즉시 시연 가능 |

## 구현한 강화학습 요소 (전부 from scratch)

DQN(MLP 8→16→16→5, He 초기화, 역전파 직접 구현) · Double DQN(토글) · **PER — 우선순위 경험 재현**(sum-tree O(log n) 샘플링, α=0.6, 중요도 샘플링 보정 β=0.5, Schaul 2016) · **n-step 리턴**(1/3/5, 절단 시 부트스트랩 유지) · Experience Replay(5만) · Target Network · ε-greedy 지수 감쇠 · Huber 손실 · Adam(모델 교체 시 옵티마이저 리셋) · 절단(truncation) vs 종료 구분 · 랜덤 스타트 · 그리디 평가 분리(학습 지표 ≠ 정책 실력) · **희소 보상 모드**(1/5바퀴 체크포인트 보상)

**환경**: 상태 = 레이캐스트 7방향 + 속도(8차원), 행동 = 조향 5단계, 보상 = 중심선 전진량 − 충돌 페널티. 60Hz 고정 타임스텝, 스윕 충돌 검사, 공간 해시 그리드.

| 고스트 랩 | 저장→더 빠른 랩 갱신→느린 랩 무시 로직 검증, 레이스 중 재생 렌더링 확인 |
| 온보딩 투어 | 5단계 스포트라이트 정확 배치·완주·영구 저장 확인 |
| PER + 3-step (메인 학습) | 14,000스텝에 최고 3.3바퀴 — v1 기본 DQN(15k에 1.35바퀴) 대비 약 2.5배 가속 |

| 게임 필 (v4.0) | 어트랙트 모드 주행, 네온 트랙 렌더, 세리머니(YOU WIN!+콘페티 130개)→결과창 전환, 충돌 3회 시 흔들림·히트스톱 동작 확인 |

## 2분 시연 시나리오 (발표용)

1. **(0:00)** "여러분은 AI를 매일 쓰지만, AI가 배우는 걸 본 적 있나요?" → MAX 학습 시작, 무작위 충돌
2. **(0:30)** 학습 곡선 우상향 + 🗺 가치 지도 켜기 — "AI의 머릿속이 색으로 그려집니다"
3. **(1:00)** 첫 완주 → 🕳️ 보상 해킹 실험: "SNS 알고리즘이 왜 자극적으로 변하는지, 30초면 재현됩니다"
4. **(1:20)** 트랙을 즉석에서 그리고 무학습 완주 — "자율주행이 모든 길을 외우지 않는 이유"
5. **(1:40)** 심사위원에게 키보드를 — 🏁 방금 태어난 AI와 대결. 끝나면 📄 리포트 자동 생성 시연

## 조작법 & 배포

- 훈련: ▶ 시작(Space), 속도 1×~MAX, 시범 주행(D), 가치 지도(H), 하이퍼파라미터/보상 슬라이더
- 트랙: 프리셋 3종 + 📅 오늘의 트랙, 드래그로 그리기, 공유 코드 / 대결: ←→(A/D)·터치, 👻 고스트, ⚔️ 관전, 🏆 리그
- 상단: 🔬 실험실 · 🎯 도전과제 · 📖 개념 도감 · 🔊 소리 · ⚙️ 설정(리포트, 백그라운드 학습, 투어, 초기화)
- 배포: 정적 파일 4개(index.html + PWA 3종) — GitHub Pages / Netlify Drop / 학교 서버 어디든. index.html 단독으로도 완전 동작
